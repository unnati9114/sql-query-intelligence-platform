# SQL Query Intelligence Platform

Production-style system for analyzing query performance, detecting anomalies, and monitoring database behavior at scale.

---

## Capabilities

* Tracks query volume, latency, and errors
* Identifies slow queries and bottlenecks
* Detects anomalies in query patterns
* Visualizes system behavior through an interactive dashboard

---

## Stack

SQL · JavaScript · Chart.js · D3.js

---

## Usage

Open `dashboard.html` or load a CSV dataset.
